Project Name

Project Overview

  • Our puropse for this project is to create an STD clinic for patients and run test for STD diseases and also educate our patients with what it is.
  • Intended users for this website are for people who want to learn more about STD's(sexually transmitted disorders), and also use resources where they can learn prevention methods on how to handle these diseases.
  • This website is going to have 7 pages which are going to be a home page, help page for appointments, information page, contact page, how to prevent STD's from occuring, testing page, and about page.

  • Audience: The audience will be people who have experienced STD's or who are curious to know what they are.
  • Content: The content of this page is the going to be defining what STDs are and a background of them, and the different types of them.
  • This page is not asking any users to enter data, its going to display information and the different types of std's.
